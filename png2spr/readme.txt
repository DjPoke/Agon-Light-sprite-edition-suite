png2spr
=======

This tool for Agon Light computers converts BMP images to sprite data.

png2spr is programmed with Purebasic 6.1.2, runs under Windows, and can
easily be compiled for Mac (intel, M1/M2), Linux and Raspberry Pi.

sprite structure:
=================
frames count : byte
spr width    : byte
spr height   : byte
data         : width x height bytes of colors
